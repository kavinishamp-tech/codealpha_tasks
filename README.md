# 📌 Task 1: Web Scraping

## 📖 Description
This project demonstrates web scraping using Python to extract data from a public website. The script collects information about books such as title, price, and rating, and stores the extracted data in a structured dataset for analysis.

## ⚙️ Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## 🚀 Features
- Extracts book information from multiple pages
- Collects title, price, and rating
- Converts ratings into numeric values
- Saves data into a CSV dataset
- Handles HTML structure and navigation automatically

# 📌 Task 4: Sentiment Analysis

## 📖 Description
This project performs **sentiment analysis** on textual data using Natural Language Processing techniques. The application analyzes user input text and classifies it into **positive, negative, or neutral sentiment**.

A simple **web interface** allows users to enter text and view sentiment results along with a visual sentiment graph.

## ⚙️ Technologies Used
- Python
- Flask
- VADER Sentiment Analysis
- HTML
- CSS
- Chart.js

## 🚀 Features
- Classifies text sentiment (Positive, Negative, Neutral)
- Uses NLP techniques for emotion detection
- Interactive web interface
- Displays sentiment scores visually using charts
- Helps analyze opinions and public sentiment trends
