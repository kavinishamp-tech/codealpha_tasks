# 📌 Sentiment Analysis Project

## 📖 Description
This project performs **sentiment analysis** on textual data using Natural Language Processing techniques. The application analyzes user input text and classifies it into **positive, negative, or neutral sentiment**.

A simple **web interface** allows users to enter text and view sentiment results along with a visual sentiment graph.

## ⚙️ Technologies Used
- Python
- Flask
- VADER Sentiment Analysis
- HTML
- CSS
- Chart.js

## 🚀 Features
- Classifies text sentiment (Positive, Negative, Neutral)
- Uses NLP techniques for emotion detection
- Interactive web interface
- Displays sentiment scores visually using charts
- Helps analyze opinions and public sentiment trends
